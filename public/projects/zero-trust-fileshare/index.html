Zero Trust File Sharing

Tech: Rust, Flutter, Libp2p, AES-256-GCM

Built a peer-to-peer file sharing application with end-to-end encryption using X3DH key agreement and Double Ratchet algorithm for forward secrecy.

Implemented a Rust backend (Tokio, Axum, Libp2p) and a cross-platform Flutter frontend, enabling secure file transfers without a central server.
